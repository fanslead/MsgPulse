# AGENTS.md

## 1. 技术栈硬约束
- 后端：.NET 10 Minimal API + EF Core
- 数据库：SQLite
- 前端：React + Next.js
- 交互：RESTful API，前后端分离
- 无鉴权、无中间件额外封装

## 2. 需求边界约束
- 多厂商 SMS / Email / APP 推送配置
- 短信模板（同步+手动）、邮件模板配置
- 消息路由规则（渠道选择）
- 消息发送记录与状态管理
- 管理端界面 + 业务调用 API
- 无租户、无 SaaS、无多企业隔离

## 3. 严禁实现内容
- 权限系统、角色、登录认证
- 租户隔离、多企业数据隔离
- 过度设计、通用框架化、无关业务逻辑

## 4. 代码风格
- 代码逻辑清晰、简练
- 禁止炫技写一堆人类难以理解的代码
- 符合对应技术栈的最佳实践
